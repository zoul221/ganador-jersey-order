## Setup

1. Clone the repository:
```bash
   git clone https://github.com/YOUR_USERNAME/ganador-webapp.git
   cd ganador-webapp
```

2. Install dependencies:
```bash
   npm install
```

3. Run the development server:
```bash
   npm run dev
```

4. Open your browser to `http://localhost:5173`

## Build for Production
```bash
npm run build
```
